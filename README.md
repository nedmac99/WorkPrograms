# WorkPrograms

A collection of programs I created to help improve efficiency at my jobsite. These tools automate repetitive tasks and streamline daily workflows.

## Programs

### Stratus Manufacture Warranty Form Automation
**Location:** `stratus_mw_form_automation/`

A Chrome extension (Manifest V3) that automates filling out internal repair forms. Features include:
- Auto-filling System Hours, Oxygen Purity, and other prompts
- Populating Diagnostic Analysis, Parts Used, and Notes fields
- Handling Ending Test Results (Flow, O2 Purity, PSI, Hours Out, etc.)
- Parts table automation with confirmation handling
- Dry Run mode for previewing actions before submission
- Local storage of values via Chrome sync

See the [Form Automation README](stratus_mw_form_automation/README.md) for setup and usage instructions.

### Parts List Tracker
**Location:** `Parts List/`

A Python script for tracking Serial numbers and associated parts. Provides a simple interface to:
- Log Serial number entries
- Record multiple parts per entry
- Save data to a persistent text file

### Weighted Output Tracker
**Location:** `weighted_output_tracker/`

A tool for tracking and analyzing weighted output data for work efficiency metrics.

## Getting Started

Each program has its own requirements. Please refer to the specific folders for detailed setup instructions.

### Prerequisites
- Python 3.x (for Python scripts)
- Google Chrome (for the Form Automation extension)
